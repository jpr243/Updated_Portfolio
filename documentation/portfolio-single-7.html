bamazon Node.js & MySQL

Project Details

  • Category : MySQL
  • Completed : December 2019
  • Deployed Version : bamazon Node.js & MySQL
  • Code on Github : bamazon Node.js & MySQL
Overview

The project brief was to make an Amazon-like storefront utilising the MySQL skills learnt. The app takes orders from customers and depletes stock from the store's inventory.


Project Details

Customer View

  • Firstly, the app asks the customer the ID of the product they would like to buy.
  • Secondly, a prompt asks the customer how many units of the product they would like to buy.
  • Once the order is placed, the app checks if the store has enough of the product to meet the request. It updates the SQL database to reflect the remaining quantity and shows the customer the total cost of their purchase.
  • The app advises the customer if there is insufficient quantity to meet their order.

Manager View

  • There are four options for the manager to choose.
  • View Products for Sale - lists every available item.
  • View Low Inventory - lists all items with an inventory count lower than five.
  • Add to Inventory - displays a prompt that will let the manager "add more" of any item currently in the store.
  • Add New Product - allows the manager to add a completely new product to the store.
