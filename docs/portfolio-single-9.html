Food Sonic

Project Details

  • Category : Group Projects
  • Completed : January 2020
  • Deployed Version : Food Sonic
  • Code on Github: Food Sonic
Overview

The requirements for the project were to:

  • Use Node and Express Web Server
  • Be backed by a MySQL Database (Sequelize)
  • Have both GET and POST routes for retrieving and adding new data
  • Be deployed using Heroku
  • Utilize at least one new library, package, or technology. We utilised Lodash, Handlebars Partials and Tailwind CSS.
  • Have a polished frontend / UI
  • Have folder structure that meets MVC Paradigm
  • Meet good quality coding standards (indentation, scoping, naming)

Project Details

Our team created the Food Sonic app:

  • This app is designed for the couple who want an intimate dinner experience dining in at Food Sonic.
  • They can sit down at our Restaurant with an iPad at their table, browse the menu, see a visual of the food and drink options and order at their table.
  • The kitchen will immediately receive a time stamped order to prepare the food and have the server deliver the order to the table.
  • No printed menus.
