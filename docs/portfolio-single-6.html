Liri-Bot App

Project Details

  • Category : nodejs
  • Completed : November 2019
  • Deployed Version : Liri-Bot App
  • Code on Github : Liri-Bot App
Overview

The project brief was to make LIRI. LIRI is like iPhone's SIRI. However, while SIRI is a Speech Interpretation and Recognition Interface, LIRI is a Language Interpretation and Recognition Interface. LIRI will be a command line node app that takes in parameters and gives you back data


Project Details

LIRI will search Spotify for songs, Bands in Town for concerts, and OMDB for movies.

  • The command concert-this will search the Bands in Town Artist Events API for an artist and render the following information about each event to the terminal.
  • The command spotify-this-song shows information about the song in the terminal.
  • The command movie-this will output information about the movie into the terminal.
  • The command do-what-it-says takes the text inside of random.txt and then use it to call one of LIRI's commands.
